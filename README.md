# Ai-rceus-Pokedex
Project completed during the second year of the B.Sc. in "Mathematical Sciences for Artificial Intelligence" for the Machine Learning course.

1. Instructions for use:
2. Download the .ipynb notebook.
3. Download the Pkmn folder from the following link: https://drive.google.com/drive/folders/1-NZo4aqBjwnOFiV_hfLXoajCUcTtAMgv?usp=sharing
4. Upload all files to your personal Google Drive.
5. Run all the cells in the notebook.

Note on Training: The code is ready for use without additional training, thanks to the checkpoints included in the attached folders. To start a new training session, simply set the value to True in the checkpoint section or remove the existing checkpoints from the folders.
Audio/Music Section: Some tracks are already available in the "Input" folder, but feel free to add your own! Please note that there is no auto-save function for the generated images.
